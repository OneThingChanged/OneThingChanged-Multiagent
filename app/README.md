# MultiAgent App

This directory contains the Tauri 2 + React + TypeScript desktop app.

## 한국어

루트 `README.md`에 프로젝트 전체 설명이 있습니다. 이 폴더는 실제 앱 소스와 빌드 설정을 담고 있습니다.

### 개발 실행

```bash
npm install
npm run tauri dev
```

### 릴리즈 빌드

```bash
npm run tauri build
```

빌드 산출물은 `src-tauri/target/release/bundle/` 아래에 생성됩니다.

## English

The root `README.md` contains the full project overview. This directory contains the actual app source code and build configuration.

### Development

```bash
npm install
npm run tauri dev
```

### Release Build

```bash
npm run tauri build
```

Build artifacts are generated under `src-tauri/target/release/bundle/`.
