# OneThingChanged MultiAgent

Tauri 2 + React + TypeScript desktop app for managing multiple AI-agent terminal sessions in one window.

## Run

```bash
cd app
npm install
npm run tauri dev
```

## Build

```bash
cd app
npm run tauri build
```

## Features

- Sidebar-managed Claude/Codex/Shell agents
- Tabbed and split terminal panes
- Drag-and-drop pane organization
- Ctrl+C copy, Ctrl+V paste, Ctrl+wheel terminal zoom
- Right-side Markdown Docs panel with List/Tree/Hide navigation
- Clickable Markdown paths from terminal output
- Global Soft/GitHub/Warm/Light themes

Project documentation lives in `docs/`.
